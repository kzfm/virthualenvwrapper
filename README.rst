=====================================
 Virtual Haskell Environment Wrapper
=====================================

What is it?
-----------

virtualenvwrapper for Haskell

Commands
--------

- mkvirthualenv
- rmvirthualenv
- lsvirthualenv

workhon
-------

add your .bashrc

   function workhon(){ source $HOME/.virthualenv/$1/.virthualenv/bin/activate; }

now, you can change your environment.
